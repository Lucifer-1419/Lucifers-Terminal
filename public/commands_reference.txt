THE 3-STEP TOKEN GENERATION HACK
---------------------------------
Users CANNOT simply run the exploit command. They must run this sequence in a single terminal session:

1. Setup Command:   ./setup_env.sh
   Action:          Configures virtual environment and downloads dependencies.

2. Connect Command: ./connect --host Lucifers_Server_v01
   Action:          Bounces proxy chain and connects to server securely.

3. Exploit Command: ./run_exploit.sh --target=kali-core
   Action:          Deploys payloads, bypasses firewall, generates TKN-XXXX.
                    (This adds +1 Token to their account, but ONLY works if 
                     steps 1 and 2 were run first in the same window).
